# Signal Detection Theory (SDT) Functions, sources and more code
This repository contains Python code to facilitate SDT data collection, parsing and analysis. The code can be found in the `/src` folder. Below, you can also find key references as well as code (mainly in Python).

# SDT references

- [Nonparametric Measures of Sensory Efficiency for Sustained Monitoring Tasks](https://journals.sagepub.com/doi/10.1177/001872087902100109)
- [Audiovisual Integration Enhances Stimulus Detection Performance in Mice](https://www.frontiersin.org/articles/10.3389/fnbeh.2018.00231/full)
